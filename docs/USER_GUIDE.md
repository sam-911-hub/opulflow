# OpulFlow User Guide

## Getting Started

### Account Creation
1. Visit the OpulFlow homepage
2. Click "Get Started Free" to create an account
3. Enter your email and password
4. Your free account is now active with access to basic features

### Dashboard Overview
The dashboard provides a quick overview of your:
- Credit balances
- Recent activity
- Quick actions

## Credit System

### Understanding Credits
OpulFlow uses a pay-as-you-go credit system:
- Purchase credits for specific services
- Credits expire after 90 days
- No subscriptions or commitments

### Credit Types
- **Lead Lookup**: Find contact information ($0.25/lead)
- **Company Enrichment**: Get company data ($0.35/profile)
- **Email Verification**: Verify email deliverability ($0.05/email)
- **AI Email**: Generate personalized emails ($0.10/email)
- **Workflow**: Run automation workflows ($0.05/run)

### Purchasing Credits
1. Navigate to the Credits tab
2. Select a credit package
3. Choose payment method (PayPal or M-PESA)
4. Complete the purchase
5. Credits are instantly added to your account

## Features Guide

### Lead Management
1. Go to the Leads tab
2. Add leads manually or import CSV
3. View and manage your leads database
4. Export leads as needed

### AI Tools
1. Navigate to the Automation tab
2. Select the AI Tools section
3. Enter your prompt or requirements
4. Generate AI-powered content for emails or scripts

### Email Sequences
1. Go to the Automation tab
2. Select the Email section
3. Create a new sequence
4. Add emails to the sequence
5. Set timing and conditions

### Workflow Automation
1. Navigate to the Automation tab
2. Select the Workflows section
3. Create a new workflow or use a template
4. Configure workflow steps
5. Execute or schedule the workflow

## Team Collaboration

### Inviting Team Members
1. Go to the Team tab
2. Enter team member's email
3. Send invitation
4. Team member receives email to join

### Permission Levels
- **Owner**: Full access to all features
- **Admin**: Management access without billing
- **Member**: Limited access based on permissions

## Account Settings

### API Configuration
1. Navigate to the Settings tab
2. Add API keys for integrations
3. Manage existing API keys

### Profile Management
1. Update your profile information
2. Change password
3. Configure notification preferences
