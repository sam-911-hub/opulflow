# OpulFlow User Manual

## Introduction

Welcome to OpulFlow, the next-generation sales intelligence platform with pay-as-you-go pricing. This manual will guide you through all the features and functionalities available to users of the platform.

## Getting Started

### Creating Your Account
1. Visit the OpulFlow homepage at [opulflow.com](https://opulflow.com)
2. Click the "Get Started Free" button
3. Enter your email address and create a password
4. Complete the registration form with your basic information
5. Verify your email address by clicking the link sent to your inbox

### Logging In
1. Navigate to the login page
2. Enter your registered email and password
3. Click "Login" to access your dashboard

### Dashboard Overview
Upon logging in, you'll see your dashboard with the following sections:
- Credit balances for different services
- Recent activity summary
- Quick action buttons
- Navigation sidebar

## Credit System

### Understanding Credits
OpulFlow operates on a credit-based system:
- Purchase credits for specific services
- Use credits as needed with no subscription
- Credits expire 90 days after purchase
- Minimum purchase is $10

### Credit Types
- **Lead Lookup Credits**: For finding contact information ($0.25/lead)
- **Company Enrichment Credits**: For company data ($0.35/profile)
- **Email Verification Credits**: For verifying emails ($0.05/email)
- **AI Credits**: For generating content ($0.10/generation)
- **Workflow Credits**: For automation runs ($0.05/run)

### Purchasing Credits
1. Navigate to the Credits tab in your dashboard
2. Browse available credit packages
3. Select the package you want to purchase
4. Choose your payment method (PayPal or M-PESA)
5. Complete the payment process
6. Credits will be instantly added to your account

### Checking Credit Balance
Your current credit balances are always visible:
- At the top of your dashboard
- In the Credits tab
- Before using any paid feature

## Lead Management

### Adding Leads
1. Go to the Leads tab in your dashboard
2. Click "Add Lead" button
3. Fill in the lead's information (name, email, company)
4. Click "Save" to add the lead to your database

### Importing Leads
1. Navigate to the Leads tab
2. Click "Import CSV" button
3. Download the template if needed
4. Prepare your CSV file with the required columns
5. Upload your CSV file
6. Review and confirm the import

### Managing Leads
- View all leads in the lead table
- Filter leads by status, date, or other criteria
- Edit lead information by clicking on a lead
- Change lead status (new, contacted, qualified, lost)
- Add notes to leads for future reference

### Exporting Leads
1. Go to the Leads tab
2. Select the leads you want to export (or export all)
3. Click "Export CSV" button
4. Save the file to your computer

## AI Tools

### Email Generation
1. Navigate to the Automation tab
2. Select "AI Tools" section
3. Choose "Email Generation"
4. Enter details about your recipient and purpose
5. Click "Generate" to create a personalized email
6. Edit the generated email as needed
7. Copy or save the email for use

### Sales Scripts
1. Go to the Automation tab
2. Select "AI Tools" section
3. Choose "Script Generation"
4. Enter information about your product and prospect
5. Click "Generate" to create a sales script
6. Customize the script to your needs
7. Save or export the script

### Call Analysis
1. Navigate to the Automation tab
2. Select "AI Tools" section
3. Choose "Call Analysis"
4. Upload your call recording or transcript
5. Wait for the AI to analyze the conversation
6. Review insights and recommendations
7. Save the analysis for future reference

## Email Sequences

### Creating a Sequence
1. Go to the Automation tab
2. Select "Email" section
3. Click "Create Sequence"
4. Name your sequence and set basic parameters
5. Add emails to your sequence
6. Set timing between emails
7. Save your sequence

### Adding Emails to a Sequence
1. Open an existing sequence
2. Click "Add Email" button
3. Create a new email or use an AI-generated template
4. Set conditions for sending (if applicable)
5. Save the email to your sequence

### Managing Sequences
- View all sequences in the Email section
- Edit existing sequences
- Duplicate sequences for similar campaigns
- Pause or activate sequences as needed
- Track sequence performance

## Workflow Automation

### Creating a Workflow
1. Navigate to the Automation tab
2. Select "Workflows" section
3. Click "Create Workflow"
4. Choose a template or start from scratch
5. Configure workflow steps and triggers
6. Test your workflow
7. Save and activate

### Using Workflow Templates
1. Go to the Workflows section
2. Browse available templates
3. Select a template that fits your needs
4. Customize the template parameters
5. Save as your own workflow

### Executing Workflows
- Run workflows manually by clicking "Execute"
- Set up automatic triggers based on events
- Schedule workflows to run at specific times
- Monitor workflow execution in real-time

## Team Collaboration

### Inviting Team Members
1. Navigate to the Team tab
2. Click "Invite Member"
3. Enter the team member's email
4. Select their role and permissions
5. Click "Send Invite"

### Managing Team Permissions
- Owner: Full access to all features and billing
- Admin: Management access without billing changes
- Member: Limited access based on assigned permissions

### Sharing Resources
- Share leads with specific team members
- Collaborate on email sequences
- Share workflow templates
- View team activity and performance

## Account Settings

### Profile Management
1. Click your profile icon in the top right
2. Select "Profile Settings"
3. Update your personal information
4. Change your password if needed
5. Save changes

### API Configuration
1. Navigate to the Settings tab
2. Select "API Configuration"
3. Generate API keys for different services
4. Manage existing API keys
5. Set usage limits for API calls

### Notification Preferences
1. Go to Settings tab
2. Select "Notifications"
3. Choose which notifications to receive
4. Set notification methods (email, in-app)
5. Save your preferences

## Troubleshooting

### Common Issues
- **Login Problems**: Reset your password or check your email for verification
- **Payment Failures**: Verify your payment method details
- **Credit Usage**: Check your transaction history for unexpected usage
- **Feature Access**: Confirm you have the necessary credits for the feature
